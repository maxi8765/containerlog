New sticker

Peel a sticker off the roll, put it on the container, then scan it. This opens a form for a written description and/or a photo of what's inside.

Look up sticker

Scan a sticker already on a container to see what was logged, with an option to edit.

Barcodes

Both scan modes also read standard 1D barcodes (EAN, UPC, Code128, Code39, Codabar), not just QR codes.

No sticker handy?

Type any code into the manual entry box under either tab to test the form without scanning.

Previous entries

The dropdown above the contents box lists everything you've written before, so repeat items can be picked instead of retyped.

Your data

Everything is stored only on this device, nothing is uploaded anywhere. Download a backup before switching phones, then upload it on the new device to restore.
